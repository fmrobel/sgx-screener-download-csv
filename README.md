# sgx-screener-download-csv
this is the skill and python command to auto download SGX screener csv fill.
## Requirements
- Existing headless Chrome running with CDP on port 18800
- `pip install playwright --break-system-packages`

## Install as OpenClaw skill
git clone https://github.com/fmrobel/sgx-screener-download-csv.git /root/.openclaw/skills/sgx-screener-download-csv
